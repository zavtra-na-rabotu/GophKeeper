# GophKeeper

GophKeeper — это клиент-серверная система для безопасного хранения и управления логинами, паролями, бинарными данными и другой приватной информацией.

## Возможности

### Сервер:
- Регистрация, аутентификация пользователей
- Безопасное хранение приватных данных
- Передача приватных данных владельцу по запросу

### Клиент:
- Создание, редактирование и удаление данных
- Шифрование приватных данных

## Типы хранимой информации
- Пары логин/пароль
- Произвольные текстовые данные
- Произвольные бинарные данные
- Данные банковских карт

Любые данные могут сопровождаться произвольной текстовой метаинформацией (например, принадлежность к веб-сайту, личности или банку, списки одноразовых кодов активации и прочее).

### Сервер
```sh
# Сборка сервера
make server

# Запуск сервера с флагами
./bin/gophkeeper-server -p 50051 -d "your_database_dsn" -j "your_jwt_secret" -l 24

# Запуск сервера с переменными окружения
export GRPC_PORT=50051
export DATABASE_DSN="your_database_dsn"
export JWT_SECRET="your_jwt_secret"
export JWT_LIFETIME_HOURS=24
./bin/gophkeeper-server
```

### Клиент
```sh
# Сборка клиента
make client

# Запуск клиента с флагами
./bin/gophkeeper-client -a "localhost:50051"

# Запуск клиента с переменной окружения
export SERVER_ADDRESS="localhost:50051"
./bin/gophkeeper-client
```
